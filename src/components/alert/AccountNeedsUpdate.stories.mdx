# Alert block: PS Accounts Module needs an update

When a new version of PrestaShop Accounts is available, the
linking process cannot be done properly. So the user should be warned with that message
and the process is blocked until the module is updated.

<Story name='Not Updated' height='220px'>
  {{
    components: {AccountNeedsUpdate},
     props: {
          installLoading: {
            default: boolean('installLoading', false),
          },
        },
    template: '<AccountNeedsUpdate :installLoading="installLoading" @updateAccounts="event" />',
     methods: {
          event: action('install'),
     },
  }}
</Story>

# Integration

_Notice: If you need to integrate the standard behavior of PS Accounts, please see the
[`PsAccounts` component in the integration section](?path=/docs/integration-psaccounts-panel--connected)._

If you need a custom integration of PS Accounts sub-components, here is the EventBusNotInstalled block:

```html
<AccountNeedsUpdate :installLoading="installLoading" @installEventBus="installEventBus" />
```
