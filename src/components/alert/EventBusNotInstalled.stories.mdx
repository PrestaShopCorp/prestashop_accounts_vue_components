# Alert block: PS Event Bus module is not installed

When the PS Event Bus module is not installed in the module manager of the backoffice, the
linking process cannot be done properly. So the user should be warned with that message
and the process is blocked until the module is installed.

<Story name='Not installed' height='220px'>
  {{
    components: {EventBusNotInstalled},
    template: '<EventBusNotInstalled @installEventBus="event" />',
     methods: {
          event: action('install'),
     },
  }}
</Story>
