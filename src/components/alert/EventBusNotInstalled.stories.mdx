# Alert block: PS Event Bus module is not installed

When the PS Event Bus module is not installed in the module manager of the
backoffice a message warns the user that the module needs to be installed.

<Story name='Not installed' height='220px'>
  {{
    components: {EventBusNotInstalled},
     props: {
          installLoading: {
            default: boolean('installLoading', false),
          },
        },
    template: '<EventBusNotInstalled :installLoading="installLoading" @installEventBus="event" />',
     methods: {
          event: action('install'),
     },
  }}
</Story>

# Integration

_Notice: If you need to integrate the standard behavior of PS Accounts, please see the
[`PsAccounts` component in the integration section](?path=/docs/integration-psaccounts-panel--connected)._

If you need a custom integration of PS Accounts sub-components, here is the EventBusNotInstalled block:

```html
<EventBusNotInstalled :installLoading="installLoading" @installEventBus="installEventBus" />
```
