# Module Alert: An alert for module needed action

When PS Accounts need a dependant module action, 
that module alert component is used to warn user.

<Story
  name='Module Alert'
  args={{

    actionText: 'Activate',
    loading: false,
    loadingText: 'The PrestaShop Event Bus module is currently being activated...',
  }}>
  {Template.bind({})}
</Story>

## Integration

```html
<ModuleAlert

  action-text ="Activate"
  :loading="false"
  loading-text="The PrestaShop Event Bus module is currently being activated..."
  @action="() => {}"
  />
```
